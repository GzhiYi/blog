use loader 需要保证顺序

模块的 loader 可以链式调用。链中的每一个 loader 会将本 loader 处理结果递给下一个。需要注意链是逆顺序执行（从右到左，或说从下至上）。

rules: [
  {
    test: /\.css$/i,
    use: ['style-loader', 'css-loader']
  }
]

上面表示所有匹配的 css 文件先给 css-loader 处理，再给 style-loader 处理。
